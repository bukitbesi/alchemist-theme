# Alchemist Theme Demo

## Overview

This is a demo version of the Alchemist Theme showing how the template grid works with sample content.

## Files Structure

- `index.html` - Demo homepage with working grid layout
- `alchemist-theme1.xml` - Original Blogger template (needs to be installed on Blogger)
- `css/theme-styles.css` - Theme styles
- `css/dark.css` - Dark mode styles
- `js/theme-scripts.js` - Theme JavaScript functionality

## How to View the Demo

1. Open `index.html` in your web browser
2. You'll see a working grid layout with sample posts
3. Try the dark mode toggle (moon icon in bottom left)
4. Test the responsive design by resizing your browser

## Features Demonstrated

- ✅ Responsive post grid (3 columns on desktop, 1 on mobile)
- ✅ Dark mode toggle with persistent state
- ✅ Mobile-friendly navigation
- ✅ Breaking news ticker
- ✅ Search bar
- ✅ Sidebar with widgets
- ✅ Back to top button
- ✅ Footer with newsletter signup

## Using on Blogger

To use the actual Blogger template:

1. Go to your Blogger Dashboard
2. Navigate to Theme > Customize
3. Click "Backup/Restore"
4. Upload `alchemist-theme1.xml`
5. The template will automatically pull your blog posts

## Customization

- Colors can be changed in the CSS variables (`:root` section)
- Grid layout can be adjusted in the `.post-grid` CSS
- Add more widgets in the sidebar section

## Note

The demo uses placeholder images from Picsum. On Blogger, these will be replaced with your actual post thumbnails.
